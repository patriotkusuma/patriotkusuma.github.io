About

Hello World!

Selamat datang di patriotkusuma.github.io.

Disini saya akan berbagi pengetahuan, pengalaman, dan aktivitas saya. Dalam blog ini juga akan ada beberapa tutorial tentang bahasa pemrograman mulai dari hal-hal dasar hingga tingkat ahli (expert), artikel tentang dunia IT, dan masih banyak lagi.

Blog ini Disusun menggunakan hugo versi 0.49. Dan menggunakan template theme casper-two dari akang EUEUNG MULYANA.

Perkenalkan nama saya Patriot Kusuma Sejati, bisa dipanggil ipat atau patriot. Saya banyak menghabiskan waktu saya di depan layar. Saya menggunakan sistem operasi berlogo tux, yang saya gunakan untuk belajar pemrograman, menulis, maupun design.

Saat ini saya masih berstatus sebagai mahasiswa aktif salah satu perguruan tinggi swasta yang ada di Yogyakarta, yaitu Universitas Amikom Yogakarta dengan prodi INFORMATIKA, salah satu prodi yang didominasi oleh kaum adam.

Sebelum nya saya bersekolah di SMA N 1 BAMBANGLIPURO. Salah satu sekolah negeri yang berada di daerah Bambanglipuro, Bantul, Yogyakarta.

Saya juga termasuk aktif di sosial media, seperti instagram dengan username @patriotkusuma, atau kalian menemukan saya di telegram dengan useranme @patriotkusuma.

Kenapa membuat blog patriotkusuma.github.io?

Alasan utama saya membuat blog adalah sebagi tempat untuk mencatat hasil belajar saya tentang pemrograman ataupun sharing hal-hal kecil seputar dunia IT ataupun aktivitas saya.

Selain itu, dengan adanya blog ini saya bisa berbagi ilmu dan pengamalam. Meskipun pengalaman dan ilmu saya masih belum banyak, tetapi tidak ada salahnya untuk berbagi hal-hal positif untuk kalian semua.

Harapan Saya

Semoga dengan kehadiran blog ini dapat bermanfaat untuk kalian semua. Terima kasih telah mendukung blog ini.
